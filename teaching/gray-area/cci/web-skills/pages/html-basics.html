The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

The <div> tag defines a division or a section in an HTML document.

The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

The <div>; tag is easily styled by using the class or id attribute.

Any sort of content can be put inside the <div> tag!



The <span> tag is an inline container used to mark up a part of a text, or a part of a document.

The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.

The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

The <strong> tag is used to define text with strong importance. The content inside is typically displayed in bold.

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.

The <title> tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

The <title> tag is required in HTML documents!

The contents of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search results.

The <title> element:

  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page in search-engine results

The <ul> tag defines an unordered (bulleted) list.

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

    <li>Tea</li>
    <li>Milk</li>
 </ol>

The <img> tag is used to embed an image in an HTML page.

Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

The <img> tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
<img src="images/cat.png">

By default, your image will appear as its original size. To resize the image, include width="value" inside the <img> tag and set the value to either a percentage like 100% or a fixed value like 100px.

Tip: To link an image to another document, simply nest the <img> tag inside an <a> tag.

Putting ../ in a file path, means that you are going backwards, out of a current folder before you enter a new one. You will use this when you are referencing urls/file paths for <a> tags or <img> tags.

If you see a 404 error or broken img icon, and you're confused, it might be because it is missing this notation to direct to the correct file in your folder structure.
