This is a collaborative freewrite activity that is meant to take place in person, in a circle. Each person responds to a prompt by writing in html for 10 minutes. And then after the round ends, everyone shifts their seat and responds to the next prompt for an other 10 minutes. This process repeats for however long the activity goes on for leaving the group with a collaborative website.

Navigate to the shared Google Drive folder > Web Skills > Handcoding Round Robin.

As you can see there are 5 rounds. Each 10 minutes.

Download this github repo. And use the example folder, your name, to code in each round.

At the end of eah round, drop your index.html file in. We will know whose is whose by the "owner" column so don't worry about renaming the file. This is just so you can copy and paste their code into your editor.

Round 1

Revisit the question, what is your first internet memory? And respond in as much detail you can to start populating your index.html file with as much text as you can.

Use the <div> tag to create sections, and write paragraphs using the <p> tag and add titles and subtitles using the <h1>, <h2>, <h3> tags to format your text. Give your page a title in the <title> tag as well.

At the end of your writing, come up with a question to ask the next person to respond to. Write it down in the html document.

Round 2

Use the <a> tag to create links to different pages on the internet by wrapping the element around words written by the person before you. This will connect oursites out to other spaces on the internet. See how many relevant sites you can find that would work as links.

Then respond to the question from the person before you and add it to the document.

Round 3

Read these two responses and start to populate the page with images by using the <img> tag. Go wild with them

Round 4

The site has probably become chaotic enough at this point. Respond with a list. Us the <ul> or <ol> tag and next <li> tags inside. And list out an array of thoughts based on what you just looked at.

Round 5

Create a form for a viewer. This is a very conceptual form and shouldn't be boring. Use the <form>, <label>, and <input> tags to create this.

Zip up your final folder with your name and drop it into the folder titled final in the Google Drive.

We will now spend the rest of the time with show and tell!
