Hello!

Helcome to gray area, and welcome to the first module of creative code intensive, web skills!

This site might look pretty bare, but that is because it is only composed of HTML – hypertext mark up language.

As we learn the make up of the web, I will enable more components of this site like styling! For now, I hope you appreciate this site in it's devoid state.

Lets go around the room and introduce ourselves a bit.

For the next three sessions, we are going to dive deep into the make up of the web and code our own small little site. And in doing so, we will start to make small little plots of our own on the internet.

Once you learn the components of what makes up the internet, coding websites can start to feel like a handmade craft. Each line being written and composed carefully. This is called the handmade web.

But to start off, a little bit of context

Oliana Lialina is a pioneer of net art born in Russia. Her network-based art includes My Boyfriend Came Back from the War (1996), Agatha Appears (1997), First Real Net Art Gallery (1998), and Last Real Net Art Museum (2000). She wrote A Vernacular Web in 2005 which talked about the old aesthetics of the World Wide Web I, the web during its amateur state.

According to Lialina, the amateur web felt boundless and hopeful. Many sites had big "under construction signs". It didn't mean that the site didn't work, it meant that someone was always there maintaining the site. Sites were playful and decorated with sparkly gifs and starry backgrounds. It was customized and expressive with cooperative effort to connect their sites to others.

In 2015, J. R. Carpenter wrote in his essay, The Handmade Web,

I evoke the term 'handmade web' to refer to web pages coded by hand rather than by software; web pages made and maintained by individuals rather than by businesses or corporations; web pages which are provisional, temporary, or one-of-a-kind; web pages which challenge conventions of reading, writing, design, ownership, privacy, security, or identity.

I evoke the term 'handmade web' in order to make a correlation between handmade web pages and handmade print materials, such as zines, pamphlets, and artists books.

So with that, I ask you two questions:

What is your first internet memory?

If you were to remake the internet by hand, how would you envision it?

Laurel Schwurlst is an artist whose focus is on the poetics of the internet. She wrote My website is a shifting house next to a river of knowledge. What could yours be?, which compares HTML to building the scaffolding of a house. She goes on to talk about a website as a metaphor – website as a room, as a shelf, as a plant, as a garden, etc.

With that said, we will begin to explore the handmade web and start by coding purely in HTML. For this you will need a code editor. We will be using Visual Studio Code. Once you download that, you should open the editor and navigate to the extensions icon on the very left panel. It should be the last icon. install these extensions:

  • HTML Boilerplate
  • htmltagwrap
  • Beautify
  • Live Server
  • Code Spell Checker

And now we are ready! Let's move on to the next part, The Command Line Tool
