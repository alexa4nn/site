HTML is like building the frame and foundation of a house and CSS is like painting and decorating each room of your home.

CSS is linked to HTML by using <link href="styles.css" rel="stylesheet">

In the folder you made in your home folder, add a styles.css file and link it to your index.html file.

Your folder structure will now look like:

  • folder name
    • index.html
    • styles.css

CSS has a specific syntax and it is made up of a few different components:

selector {
    property: value;
}

For example, if you want to make all of the text in the <body> element 72 pixels: 

body {
    font-size: 72px;
}

All of the text will now be 72 pixels

Lets try some other styling

body {
    font-size: 24px;
    background-color: blue;
    color: hotpink;
}

The block of code that lists out all of the properties are called a declaration block. You can think of this as a list of rules.

All of your body text will now be 24 px, hotpink and have a background of blue.

You can style specific html elements by noting the element name as a selector.

a {
    font-size: 36px;
    color: limegreen;
    text-decoration: none;
}

All of your body text will now be 24 px, hotpink and have a background of blue. And a hyperlink will be green with no underline.

Selectors also can have "pseudoclasses" that enact styling on special occasions. For example, on hover. 

a :hover {
    font-size: 36px;
    color: aqua;
    text-decoration: none;
}

All of your body text will now be 24 px, hotpink and have a background of blue. And a hyperlink will be lime green with no underline and turns aqua when you over on it.

You can do this with any element!

div {
    background-color: aquamarine;
    height: 100px;
}
div:hover {
    background-color: rgb(255, 30, 0);
}

This is a div that has a height of 100px and changes color on hover.

Notice how you only need to notate which rules change on hover. You do not need to repeat code if it is the same for both.

You can also make divs that have images that change on hover! You do this with the background-image property.

img {
    background-image: url(images/cat.png);
    height: 200px;
    width: 200px;
    background-size: contain;
    background-repeat: no-repeat;
                }
img:hover {
    background-image: url(images/bearhug.png);
}

In order to specify elements or group together elements, there is an additional way to identify one or class elements together using the id=" " or class=" " properties inside of your html tags.

So if you have something like <div id="frog"><div> or <div class ="lizard"></div> in your html, you would notate it in CSS like:

#frog{
    property: value;
}

or

.lizard{
    property: value;
}

Notice that id's use # before their name and classes use . before their name.

There are two ways to import fonts into your page. One is to use an api like google fonts. Which is simpler but needs internet to connect. And you are bound to the repository of that database of fonts.

When you click on a font you like, select the font style.

It will say something like "Select Regular 400". This is saying that you are selecting a particular font style and weight from the font family.

Navigate to the very top right of the page. When you hover over the icon, it says "View Selected Families".

Scroll below and you will see code. One box is HTML and the other is CSS. Copy the HTML in the <head> tag.

Then copy the CSS in the correct block of CSS. If it is the main font for the site, stick it in the <body> tag.

Downloading fonts and using them, we use the <@font-face> selector.

All you need to do is make a fonts folder and place your webfont files inside. These files might end with <.woff>, <.woff2>, and/or<.otf>

You would notate this in CSS like:

@font-face{
    font-family: Pentameter Regular;
    src: url(fonts/Pentameter-Wide-
    Regular-Trial.woff);
}
body{
    font-family: 'Pentameter Regular';
}

Positioning will become an important component of CSS for if you want to make layouts for anything other than a one column page with divs stacked on top of each other. Right now, the default display property is display: block which means stacked on top of each other.Most elements are set to this by default. And some elements are display: inline; by default, which means side by side.

display: block

display:inline

----- -----

But you could position things that differ from their "normal" position. by using the position property.

position: absolute; is positioned relative to the nearest positioned ancestor.

position: relative; is positioned relative to its normal position.

position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.

For every element you use the position property, you need to use top, left, bottom, and right to position it on the page.

#swan{
    position: absolute;
    top: -200px;
    right: 1050px;
}
