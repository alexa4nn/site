Helpful Terms

The best folder naming practice is all lowercase filenames and folder name and no spaces. Use dashes or underscores. for example: my_file.txt or my-file.txt.

try to navigate through the garden-of-forking-paths folder by only using the command line tool.

Then navigate back to your home folder by typing cd .., and make a new directory by typing mkdir foldername. Name the folder whatever you would like. This will hold your site that you will create this week and any other files you create. Create an html file by typing touch index.html.

If you want to run VS Code from the terminal by simply typing 'code', VS Code has a command, Shell Command: Install 'code' command in PATH, to add 'code' to your $PATH variable list. In VS Code, press Cmd + Shift + P and type shell command to find the Shell Command: Install 'code' command in PATH command. After executing the command, restart the terminal for the new $PATH value to take effect. You'll be able to simply type 'code .' in any folder to start editing files in that folder.

In terminal you can simply navigate to your folder with cd foldername and open it in VS Code with code foldername.
